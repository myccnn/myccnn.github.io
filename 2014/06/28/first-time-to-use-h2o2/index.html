第一次清创

上周四骑车摔了一跤，原因嘛就不说了，好在戴了头盔，没什么大碍，就是右侧肩膀、胳膊肘、腰和膝盖严重擦伤。第二天起床，发现摔的部位挺疼，而且擦伤的地方一直在渗血，所以向主管请了假，准备去医院检查一下。

到了医院，挂了外科，医生问了下情况，让去拍片子，主要是胸腔和骨盆，拍完片子再去清创。到了清创的医疗室，里面有几个护士。一个大妈护士让我坐下，然后问我伤了哪儿。我指了指前面说的四个地方，大妈用方言说，你这小伙子摔得不轻啊。说着，大妈就开始准备清创用的各种溶液和器具。

用的溶液是双氧水，因为我看到瓶子上写着H2O2。刚想问大妈会不会痛，大妈直接就把沾了双氧水的棉棒凑到伤口上。O!M!G! F!!! S!!! F!!! S!!! 此处略去脏话不表。那种疼痛无法用语言表达，唯一能说出口就是“啊～～啊～”。虽然想忍住，但是真的忍不住啊！因为伤口有脏东西，大妈还用力擦！疼痛加倍，疼得我感觉都麻了……

清创完毕，我感觉我的右半边身子像失去了知觉一样。身后大妈还朝我喊道：“不好再来啊！”“……”

回到家，我瞬间对关二爷崇拜有加。想当年，关二爷不用麻沸散就刮骨疗伤，用华佗的话来说，真乃神人也。如果真有此事，那我简直弱爆了。
